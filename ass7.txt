Implementation – Part A: S3 Trigger
 Step 1: Create S3 Bucket
1.	Go to AWS Console 
2.	Open S3 
3.	Click Create Bucket 
4.	Enter unique name (e.g., my-upload-bucket) 
5.	Keep default settings → Create 

Step 2: Create Lambda Function
1.	Go to Lambda 
2.	Click Create Function 
3.	Choose Author from scratch 
4.	Function name: S3TriggerFunction 
5.	Runtime: Python 3.x 
6.	Click Create
 
Step 3: Write Lambda Code
import json

def lambda_handler(event, context):
    print("Event Triggered:", json.dumps(event))
    
    for record in event['Records']:
        bucket = record['s3']['bucket']['name']
        file_name = record['s3']['object']['key']
        
        print(f"File uploaded: {file_name} in bucket: {bucket}")
    
    return {
        'statusCode': 200,
        'body': 'S3 Trigger Executed Successfully'
    }
click deploy

Step 4: Add S3 Trigger
1.	In Lambda → Add Trigger 
2.	Select S3 
3.	Choose your bucket 
4.	Event type: PUT (Object Created) 
5.	Enable trigger
 
Step 5: Test
•	Upload a file to S3 bucket 
•	Go to Lambda → Monitor → Logs 
•	Check output in CloudWatch 
 Implementation – Part B: HTTP Trigger (API Gateway)

Step 1: Create Lambda Function
•	Name: HttpTriggerFunction 
•	Runtime: Python 

Step 2: Write Code
def lambda_handler(event, context):
    name = event.get('queryStringParameters', {}).get('name', 'Guest')
    
    return {
        'statusCode': 200,
        'body': f"Hello, {name}! Welcome to Lambda HTTP trigger."
    }

Step 3: Create API Gateway
1.	Go to API Gateway 
2.	Click Create API 
3.	Choose HTTP API 
4.	Add integration → Lambda 
5.	Select your function 
6.	Deploy API 
 
Step 4: Get Endpoint URL
Example:
https://abc123.execute-api.region.amazonaws.com

Step 5: Test API
Open browser:
https://your-api-url?name=abc
Output:
Hello, abc! Welcome to Lambda HTTP trigger
